# Console Catch Game (C#)

## Açıklama
Bu proje C# ile geliştirilmiş console tabanlı bir skor toplama oyunudur.

## Oyun Özellikleri
- Oyuncu '@' karakteri ile temsil edilir
- Ok tuşları ile hareket edilir
- Yukarıdan '*' ve 'O' nesneleri düşer
- Yakalanan nesneler skor kazandırır

## Ek Özellikler
- Level sistemi
- Pause (P tuşu)
- Süre bazlı bitiş
- High score sistemi

## Log Sistemi
Program çalışırken:
- INPUT
- MOVE
- UPDATE
- COLLISION
- GAME OVER

olayları log.txt dosyasına yazılır.

## Çalıştırma
dotnet run
